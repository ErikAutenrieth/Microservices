## Architecture


- **Repository Management:** Nx (Monorepository)
- **Programming Language:** TypeScript
- **Frontend:** Angular
- **Microservices:** Express/NestJS Server
- **Workflow Management:** Kafka
- **Real-time Communication:** socket.io
- **Database:** MongoDB
- **Virtualization:** Docker (Swarm)
- **Monitoring:** InfluxDB
